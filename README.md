Customer Segmentation & Churn Pattern Analytics in European Banking

This project analyzes European banking customer data using Python to identify customer segments and understand churn patterns for improving customer retention strategies.

Tools Used:
• Python • Pandas • Matplotlib • Seaborn • Streamlit

Key Analysis:
• Overall Customer Churn Rate
• Churn by Country and Gender
• Age Group & Tenure Analysis
• Customer Segmentation by Credit Score and Balance
• High-Value Customer Churn Analysis
• Product Usage & Active Member Analysis

The analysis helps identify high-risk customer segments, understand customer behavior across European regions, and support data-driven retention strategies for retail banks.
